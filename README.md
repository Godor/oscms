#关于oscms开源系统
 
#### 项目目标
 
- 打造一个集内容，社区，商城的多用户，多终端的开源Web开源系统。目标客户集中在自媒体运营客户。
- 团队成员学习新知识，完善新知识，提高整体技能
- 模块，插件商店盈利。
- 提升个人执行力，不做光说不做的loser.
 
#### 系统特点
 
开源，轻量，模块化，插件化

#### 功能系统

- 基础系统
- 用户系统
- 内容系统
- 社区系统
- 电商系统
- 模块系统
- 插件系统

 

#### 团队管理
> 核心团队成员定义在项目初期为项目作出贡献的，在项目中期为项目作出突出贡献的成员。
> 核心团队成员拥有一定的权益和义务，对于不作为的核心团队成员，集体表决退出机制。
 
- 积极参与团队工作，弹性工作制度
- 会议和规则一旦制度积极遵守执行
- 对项目所有内容保密
- 积极参与促进项目发展的其他事务
